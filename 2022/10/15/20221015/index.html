基本数据类型

基本类型

number, string, boolean, undefined, null, symbol

let str1:string = "1"
let bool:boolean = false
let num1:number = 2
let und:undefined = undefined
let num:null = null
let syl:symbol = symbol("123")

ts中let与const的区别是什么
const num = 1 相当于定义了一个常量，不能修改
let 类型由值的类型决定
const 类型就是值，值 = 类型

void:

ts函数中没有返回值，函数类型是void

function obj1():void{}
function obj2():undefined{return undefined}

never

没有值，永远不会返回结果，用来返回错误。

function fn(): never {
	throw new Error('报错！')
}

特殊数据类型

any

表示任意数据类型。（一般不推荐使用）会跳过类型校验

unknown

unknown未知类型的值，类似 any,是一个类型安全的any，但是不能直接赋值给其他变量。

数组 array

数组中的每一项必须是同一个数据类型

// 数组两种写法
const arr1:number[]   //规定数组内的元素为数值类型
const arr2:Array<string>   //泛型写法，类型参数化

元组 tuple

一个数组中保存多个类型的数据
数组中的值与元祖类型必须：数量要一致 位置要对应 类型要对应

const a: [number, string] = [1,'hello']

对象 object

定义对象时，要定义出对象中有哪些属性，以及每个属性的值是什么类型

枚举 enum

用于取值被限定在一定范围内的场景，比如一周只有七天，颜色限定为红绿蓝等

enum statusCode{
	success = 200
	paramsError = 400,
	serverError = 500
}
let code:number
if(code == statusCode.success){
	console.log("成功")
}else if(code == statusCode.paramsError){
	console.log("参数错误")
}else if(code == statusCode.serveError){
	console.log("服务器错误")
}
```   
#### 数字枚举
数字枚举有自增长的行为，可以使用初始器指定从几开始自增
```js
enum test {
	//a 使用默认值为 0，其余成员会从 0 开始自动增长。
    a,	// a = 0
    b,	// b = 1
    c,	// c = 2
    d,	// d = 3
};

字符串枚举

字符串枚举没有自增的行为，每个成员必须初始化赋值

enum test {
    a = 'a',
    b = 'b',
    c = 'c',
    d = 'd',
}

异构枚举

混合字符串和数字成员，不推荐使用
